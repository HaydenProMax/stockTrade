版本时间：2026-05-30 22:46:08

## 1. 本版核心变化

本版基于 v3 的待确认事项，补齐实际执行规则。

核心确认：

```text
红利低波：长期储蓄底仓固定投一部分 + 回撤触发再加一部分
纳指每日10元：长期保留
纳指021778：400元/月必须买，未触发策略则月末择机买入
日经225：两只长期买，因为存在限额
全球科技互联：优先买A类
全组合硬上限：5500元/月
未执行额度：顺延到下一个交易日
减仓规则：除红利低波外，其他基金总投入超过10000元前不允许减仓
飞书推送：中午12:00、下午14:30
数据失败：先重试，超过3次后降级使用缓存
交易日规则：按中国交易日
飞书签名：启用
```

## 2. 总体资金框架

全组合月投入硬上限：

```text
5500元/月
```

按每月约20个交易日估算：

| 类型 | 资产组 | 月投入计划 |
|---|---|---:|
| 固定定投 | 纳指100 | 约600元 |
| 固定定投 | 日经225 | 约400元 |
| 固定+策略 | 红利低波50 | 2000元 |
| 策略/月末补投 | 纳指100 021778 | 400元 |
| 策略 | 中证A500 | 500元 |
| 策略 | 恒生科技 | 500元 |
| 策略 | 全球科技互联 | 1000元 |

合计计划：

```text
约5400元/月
```

预算缓冲：

```text
5500元硬上限 - 5400元计划投入 = 100元缓冲
```

说明：

```text
若当月交易日多于20天，固定定投金额可能推高总投入。
系统需要实时计算当月已用金额，避免突破5500元硬上限。
```

## 4. 红利低波50执行规则

红利低波作为长期储蓄资产，采用：

```text
底仓固定投一部分 + 回撤触发再加一部分
```

月预算：

```text
2000元/月
```

建议拆分：

```text
固定底仓：1000元/月
策略加仓：1000元/月
```

固定底仓执行：

```text
每月固定投入1000元
可在月初或月度再平衡日执行
```

策略加仓执行：

```text
月策略额度：1000元
1U = 250元
最大月策略加仓：4U
```

红利低波回撤触发：

```text
250日高点回撤 0% ~ -5%        不触发策略加仓
250日高点回撤 -5% ~ -10%      触发 1U = 250元
250日高点回撤 -10% ~ -15%     触发 2U = 500元
250日高点回撤 -15% ~ -25%     触发 3U = 750元
250日高点回撤 < -25%          触发 4U = 1000元，不再继续放大
```

## 5. 纳指100执行规则

纳指100由两部分组成：

```text
每日固定定投：040046、014978、270042
月度策略/补投：021778
```

### 5.1 每日固定定投

```text
040046：10元/中国交易日
014978：10元/中国交易日
270042：10元/中国交易日
```

### 5.2 021778 月度400元规则

月预算：

```text
400元/月
```

策略单位：

```text
1U = 100元
最大月投入：4U = 400元
```

正常策略执行：

```text
若回撤策略触发，则按触发金额买入021778。
```

未触发策略时的补投规则：

```text
如果当月策略未触发，或021778本月未买满400元：
    在每月最后几个中国交易日观察
    若出现下跌日，则一次性买入剩余额度
```

下跌日定义：

```text
纳指100代理指数当日涨跌幅 < 0
```

若最后几个交易日均未出现下跌：

```text
在当月最后一个中国交易日买入剩余额度
```

## 9. 回撤分层规则

回撤计算：

```text
回撤 = 当前指数点位 / 最近250个交易日最高点 - 1
```

### 9.1 低波动资产

适用：

```text
红利低波50
沪深300
```

规则：

```text
250日高点回撤 0% ~ -5%        不额外加仓
250日高点回撤 -5% ~ -10%      触发 1U
250日高点回撤 -10% ~ -15%     触发 2U
250日高点回撤 -15% ~ -25%     触发 3U
250日高点回撤 < -25%          触发最高档，不再继续放大
```

### 9.2 中波动资产

适用：

```text
中证A500
日经225
标普500
```

规则：

```text
250日高点回撤 0% ~ -7%        不额外加仓
250日高点回撤 -7% ~ -12%      触发 1U
250日高点回撤 -12% ~ -18%     触发 2U
250日高点回撤 -18% ~ -28%     触发 3U
250日高点回撤 < -28%          触发最高档，不再继续放大
```

### 9.3 高波动资产

适用：

```text
纳指100
恒生科技
全球科技互联
```

规则：

```text
250日高点回撤 0% ~ -10%       不额外加仓
250日高点回撤 -10% ~ -15%     触发 1U
250日高点回撤 -15% ~ -25%     触发 2U
250日高点回撤 -25% ~ -35%     触发 3U
250日高点回撤 < -35%          触发最高档，不再继续放大
```

## 10. 趋势过滤

回撤触发后，根据均线位置调整实际执行金额。

```text
价格 > 60日均线：
    不执行，只提示观察

价格 <= 60日均线，且 > 120日均线：
    执行 50% 触发金额

价格 <= 120日均线，且 > 250日均线：
    执行 100% 触发金额

价格 <= 250日均线：
    执行 100% 触发金额
    但不因为跌破250日均线额外加码
```

## 11. 顺延规则

未执行额度顺延到下一个中国交易日。

适用场景：

```text
基金单日限额导致无法买满
全组合硬上限暂时不足
当日数据异常导致不执行
策略建议金额大于当日允许执行金额
```

顺延限制：

```text
顺延额度仍然归属原资产组
不能自动转投其他资产组
顺延后仍受月预算和5500元全组合硬上限限制
```

## 12. 减仓规则

当前阶段默认不做减仓。

特殊规则：

```text
除红利低波之外的其他基金，总投入超过10000元之前，不允许减仓。
```

解释：

```text
红利低波作为长期储蓄资产，不纳入该减仓门槛。
纳指、日经、中证A500、恒生科技、全球科技互联等资产，在累计投入未超过10000元前，只做定投和策略买入，不做卖出。
```

## 13. 系统实现确认

### 13.1 飞书推送时间

```text
中午推送：12:00

```

### 13.2 执行记录假设

用户会严格按照策略执行。

系统默认：

```text
只要推送了建议执行金额，即默认该金额已买入。
```

因此系统需要自动记录：

```text
建议日期
资产组
基金代码
建议金额
默认执行金额
执行状态：assumed_executed
```

### 13.3 数据失败处理

数据抓取失败时：

```text
先重试
超过3次仍失败
降级使用缓存数据
```

缓存数据推送必须标注：

```text
当前信号使用缓存数据，可能不是最新行情。
```

### 13.4 交易日规则

第一版统一使用：

```text
中国交易日
```

说明：

```text
即使海外市场开盘，只要不是中国基金可交易日，就不执行买入建议。
```

### 13.5 飞书签名

飞书机器人启用签名校验。

环境变量：

```text
FEISHU_WEBHOOK_URL=飞书机器人webhook
FEISHU_WEBHOOK_SECRET=飞书机器人签名密钥
```
