## 1. 本版核心变化

本版在 v2 的基础上加入实际资金计划，并将投资行为拆成两类：

```text
固定定投池：按日或按月固定投入，不由回撤信号决定是否触发
策略加仓池：按回撤分层、趋势过滤、预算限制来决定投入节奏
```

因此，`U` 不再作为全组合通用金额，而是作为每个资产组内部的策略强度单位。

## 2. 总体执行原则

```text
1. 同一底层资产合并成一个资产组管理
2. 固定定投和策略加仓分开记录
3. 每个资产组单独设置月预算
4. 每只基金可以设置单日限额和单月限额
5. QDII、限购基金优先遵守基金自身限购
6. 未执行额度不自动转投其他资产组
7. 科技成长类资产合并看总暴露
```

## 7. 回撤分层规则

回撤计算：

```text
回撤 = 当前指数点位 / 最近250个交易日最高点 - 1
```

### 7.1 低波动资产

适用：

```text
红利低波50
沪深300
```

规则：

```text
250日高点回撤 0% ~ -5%        不额外加仓
250日高点回撤 -5% ~ -10%      触发 1U
250日高点回撤 -10% ~ -15%     触发 2U
250日高点回撤 -15% ~ -25%     触发 3U
250日高点回撤 < -25%          触发 3U，不再继续放大
```

### 7.2 中波动资产

适用：

```text
中证A500
日经225
标普500
```

规则：

```text
250日高点回撤 0% ~ -7%        不额外加仓
250日高点回撤 -7% ~ -12%      触发 1U
250日高点回撤 -12% ~ -18%     触发 2U
250日高点回撤 -18% ~ -28%     触发 3U
250日高点回撤 < -28%          触发 3U，不再继续放大
```

### 7.3 高波动资产

适用：

```text
纳指100
恒生科技
全球科技互联
```

规则：

```text
250日高点回撤 0% ~ -10%       不额外加仓
250日高点回撤 -10% ~ -15%     触发 1U
250日高点回撤 -15% ~ -25%     触发 2U
250日高点回撤 -25% ~ -35%     触发 3U
250日高点回撤 < -35%          触发 3U，不再继续放大
```

## 8. 趋势过滤

回撤触发后，根据均线位置调整实际执行金额。

```text
价格 > 60日均线：
    不执行，只提示观察

价格 <= 60日均线，且 > 120日均线：
    执行 50% 触发金额

价格 <= 120日均线，且 > 250日均线：
    执行 100% 触发金额

价格 <= 250日均线：
    执行 100% 触发金额
    但不因为跌破250日均线额外加码
```

## 9. 执行金额计算

策略加仓池最终执行金额：

```text
理论金额 = 触发U × 资产组1U金额

实际金额 = min(
    理论金额,
    资产组本月剩余额度,
    基金单日剩余额度,
    基金单月剩余额度,
    全组合本月剩余额度
)
```

未执行金额处理：

```text
同资产组内还有其他可买基金：
    可分摊到同资产组其他基金

同资产组无可买基金：
    记录为未执行
    不自动转投其他资产组
```
