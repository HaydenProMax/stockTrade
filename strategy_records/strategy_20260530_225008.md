## 2. 全球科技互联 A 类优先规则

```text
006373：A类，优先买入
021842：非优先份额，作为补充买入
```

月预算：

```text
1000元/月
```

单日限额：

```text
006373：100元/日
021842：100元/日
```

执行优先级：

```text
第一优先级：006373
第二优先级：021842
```

执行规则：

```text

    优先买006373，单日最多100元

    剩余额度可买021842，单日最多100元

如果两只基金当日限额都用完：
    剩余额度顺延到下一个中国交易日
```

## 3. 纳指100 021778 月末补投规则

纳指100 021778：

```text
月预算：400元/月
策略单位：1U = 100元
```

正常执行：

```text
若回撤策略触发，则按策略金额买入021778。
```

月末补投窗口：

```text
每月最后 5 个中国交易日
```

补投规则：

```text
如果021778本月未买满400元：
    进入最后5个中国交易日后开始观察

如果最后5个中国交易日内出现纳指100代理指数下跌日：
    在第一个下跌日一次性买入剩余额度

如果最后5个中国交易日均未出现下跌日：
    在当月最后一个中国交易日买入剩余额度
```

下跌日定义：

```text
纳指100代理指数当日涨跌幅 < 0
```

## 4. 缓存过期规则

系统数据失败时先重试：

```text
最多重试 3 次
```

超过 3 次仍失败后，允许降级使用缓存。

### 4.1 策略信号行情缓存

用于生成买入建议的最新行情缓存：

```text
最多允许过期 1 个中国交易日
```

规则：

```text
如果缓存数据日期 = 最近一个中国交易日：
    可以使用缓存生成策略信号
    推送中必须标注“使用缓存数据”

如果缓存数据落后 1 个中国交易日：
    可以给观察信号
    但不生成新增策略买入建议

如果缓存数据落后超过 1 个中国交易日：
    不生成策略建议
    只推送数据异常提醒
```

### 4.2 固定定投

固定定投不依赖行情信号，因此不受策略行情缓存失败影响。

适用：

```text
纳指100每日定投：
040046：10元/中国交易日
014978：10元/中国交易日
270042：10元/中国交易日

日经225每日定投：
020712：10元/中国交易日
020713：10元/中国交易日
```

### 4.3 策略型买入

以下策略型买入依赖有效行情数据：

```text
红利低波50策略加仓
纳指100 021778月末补投
中证A500策略加仓
恒生科技策略加仓
全球科技互联策略加仓
```

当行情缓存过期超过允许范围时：

```text
暂停新增策略买入建议
只推送数据异常提醒
不默认执行买入
```
